SwiftML

If there exists a dataset, there is an algorithm for it.











SwiftMl is a high level abstraction tool, developed to automate end-to-end machine learning model creation and deployment. This tool will analyze user-provide datasets and select the most suitable algorithm for prediction. Give us the dataset and leave the preprocessing, feature selection, algorithm selection, model training, evaluation and deployment for us to do. SwiftML is designed for easy machine learning (ML) model development without the need for coding.

Getting Started

  1. Hover to the PROCESS DATA section.
  2. Upload your csv dataset.
  3. Choose your target variable.
  4. Remove metadata columns if there are any.
  5. Choose the type of your problem (Regression/Classification).
  6. Click "Submit" to start the process.
